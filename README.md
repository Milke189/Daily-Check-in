# Daily Check-in

一个简洁的桌面每日签到计数器，支持多项目独立追踪习惯养成。

## 功能

- **多项目管理** — 同时追踪多个习惯（运动、阅读、早起等），每个项目独立计数
- **签到统计** — 累计天数、当前连续、最长连续
- **日历视图** — 按月查看签到记录，已签到日期绿色高亮
- **项目管理** — 新建、重命名、删除项目（右键标签）
- **数据持久化** — 签到数据保存在本地 JSON 文件

## 运行

仅依赖 Python 标准库，无需安装第三方包。

```bash
# 直接运行（带命令行窗口）
python checkin.py

# 后台运行（无命令行窗口，Windows）
双击 checkin.pyw
```

要求 Python 3.6+。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+R` | 重置当前项目签到数据 |

## 数据文件

签到数据保存在程序同目录下的 `checkin_data.json`，格式如下：

```json
{
  "projects": {
    "运动": {
      "total": 30,
      "streak": 7,
      "max_streak": 15,
      "last_checkin": "2026-05-05",
      "history": ["2026-05-05 08:30:00", "2026-05-04 09:00:00"]
    },
    "阅读": {
      "total": 10,
      "streak": 3,
      "max_streak": 10,
      "last_checkin": "2026-05-05",
      "history": ["2026-05-05 22:15:00"]
    }
  },
  "current": "运动"
}
```
